## Ready-to-paste draft after deposit

The data and code generated for this study are available in the archived release of this repository at the public release URL or DOI assigned at deposit. The release includes verified reference-sample tables, feature matrices, configuration files, run logs, result tables, leakage audits, variogram/block-distance outputs, figure files, and package manifests/checksums. Third-party Earth-observation products, including ESA WorldCover, Google Dynamic World, Sentinel data, AlphaEarth embeddings, and Forest Data Partnership layers, are not redistributed as raw source products; they are accessed from their original catalogues as described in the manuscript and configuration files. Very-high-resolution imagery used for visual interpretation is not redistributed unless separately licensed by the relevant provider.
