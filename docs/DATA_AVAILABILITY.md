# Data Availability Statement

The data and code supporting the manuscript are publicly available at the stable GitHub commit URL cited in the manuscript Data Availability Statement. The repository includes verified reference-sample tables, feature matrices, configuration files, run logs, result tables, leakage audits, variogram/block-distance outputs, point-level public Sentinel-2 imagery-date audit artifacts, figure files, and package manifests/checksums.

Third-party Earth-observation products, including ESA WorldCover, Google Dynamic World, Sentinel data, AlphaEarth embeddings, and Forest Data Partnership layers, are not redistributed as raw source products. They are accessed from their original catalogues as described in the manuscript and configuration files. VHR imagery used for visual interpretation is not redistributed unless separately licensed by the relevant provider.

The blinded second-reader VHR adjudication outputs are label-QA artifacts provided separately in the submission supplementary reproducibility package. They are not part of this public GitHub release. The primary manuscript metrics were computed on the locked 622-row reference table and should be rerun before using the adjudicated included subset as a replacement metric base.
