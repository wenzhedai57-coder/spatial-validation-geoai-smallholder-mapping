# Second-reader VHR adjudication outputs

These files summarize the completed blinded second-reader VHR interpretation and adjudication workflow. They are label-QA artifacts, not a recomputation of the primary manuscript metrics. The primary model metrics in the manuscript were computed on the locked 622-row reference table. Any use of the adjudicated included subset as a replacement metric base requires rerunning the full pipeline.

Boundary: VHR/image interpretation and adjudication only; weak global products are not ground truth.
