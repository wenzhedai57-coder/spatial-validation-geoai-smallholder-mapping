# Point-Level Public Imagery-Date Audit

This repository includes a point-level public Sentinel-2 acquisition-date audit for the 622 verified reference samples:

- `results/point_imagery_dates/point_level_public_sentinel2_imagery_dates_20260616.csv`
- `results/point_imagery_dates/point_level_public_sentinel2_imagery_dates_summary_20260616.json`

The audit was generated by `scripts/audit_public_sentinel2_point_imagery_dates.py`.

## Scope

For each reference point, the script queries public Sentinel-2 Level-2A STAC metadata for the 2024 analysis year and records:

- the number of point-intersecting Sentinel-2 L2A items in 2024;
- the first and last public Sentinel-2 acquisition datetime in 2024;
- the selected public Sentinel-2 item date, item ID, platform, MGRS tile, and item-level cloud cover.

The selected item is the 2024 Sentinel-2 L2A scene that intersects the point and has the lowest item-level `eo:cloud_cover`; ties are resolved by earliest acquisition datetime.

## Interpretation Boundary

This audit does not create field validation data and does not recover exact very-high-resolution basemap acquisition dates. It is a public remote-sensing metadata audit that shows each reference point has an independently traceable 2024 Sentinel-2 observation history.

Use this wording in the manuscript if needed:

`Point-level public imagery-date coverage was audited using Sentinel-2 Level-2A STAC metadata for the 2024 analysis year. For each verified reference sample, we recorded the first, last, and lowest-cloud point-intersecting Sentinel-2 acquisition date. These dates document public Sentinel-2 observation coverage and are not field-validation dates or exact VHR interpretation-platform acquisition dates.`
