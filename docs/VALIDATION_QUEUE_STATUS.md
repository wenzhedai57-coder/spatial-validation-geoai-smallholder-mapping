# Validation Queue Status

Targeted-validation candidate queues are not final reference labels and are not included as main data in this repository.

They were used only as review planning and provenance artifacts. They must not be used for metric computation, reference-label claims, or ground-truth expansion unless a real human VHR or field-verification step is completed and written to a new verified reference artifact with provenance.

The original frozen submission package retains these queue files for audit history. This GitHub-ready staging repository keeps the primary data focused on the active verified reference table and active reproducible results.
