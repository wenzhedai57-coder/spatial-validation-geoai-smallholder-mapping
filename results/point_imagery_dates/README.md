# Point Imagery Dates

This folder contains a public Sentinel-2 metadata audit for point-level imagery-date coverage.

- `point_level_public_sentinel2_imagery_dates_20260616.csv`: point-level audit table.
- `point_level_public_sentinel2_imagery_dates_summary_20260616.json`: run summary, inputs, source endpoint, counts, and limitations.
- `RUN_LOG_point_imagery_dates_*.txt`: exact run log for the generated audit.

These files document public Sentinel-2 observation dates only. They are not field-validation records and are not exact VHR basemap acquisition dates.
