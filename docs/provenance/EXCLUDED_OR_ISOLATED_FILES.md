# Excluded Or Isolated Files

The following file families are intentionally not included as main data in this GitHub-ready staging repository:

- targeted-validation candidate queues and reconstructed review queues
- files with draft/template/edit-placeholder or author-confirmation semantics
- older 619-row advisor reference tables and advisor feature stacks
- repair-history manuscript files whose names include older manuscript-family or repair-history markers

These files are not ground truth and must not be used for metric computation. The original frozen submission package retains them as audit/provenance records where applicable.
