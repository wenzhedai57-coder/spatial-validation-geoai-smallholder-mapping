# Data Availability Statement

The data and code supporting the manuscript are publicly available at the stable GitHub commit URL cited in the manuscript Data Availability Statement. The repository includes verified reference-sample tables, feature matrices, configuration files, run logs, result tables, leakage audits, variogram/block-distance outputs, point-level public Sentinel-2 imagery-date audit artifacts, figure files, the 20260624 submission supplementary reproducibility artifacts, and package manifests/checksums.

Third-party Earth-observation products, including ESA WorldCover, Google Dynamic World, Sentinel data, AlphaEarth embeddings, and Forest Data Partnership layers, are not redistributed as raw source products. They are accessed from their original catalogues as described in the manuscript and configuration files. VHR imagery used for visual interpretation is not redistributed unless separately licensed by the relevant provider.

The blinded second-reader VHR adjudication outputs are label-QA artifacts included under `results/supplementary_reproducibility_20260624/`. They do not replace the locked 622-row reference table used for the primary manuscript metrics. The primary manuscript metrics were computed on the locked 622-row reference table and should be rerun before using the adjudicated included subset as a replacement metric base.
